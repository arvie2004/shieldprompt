ShieldPrompt
Group 1 | IAS101 

Project Description
ShieldPrompt is an AI-powered security layer designed to detect prompt injections and jailbreak attempts in real-time. By utilizing both a Scikit-learn baseline and a fine-tuned DistilBERT transformer model, it provides robust protection for LLM-integrated applications.

Setup Instructions
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/shieldprompt.git`
2. Install dependencies: `pip install -r backend/requirements.txt`
3. Download the Large Model: `https://drive.google.com/drive/folders/11SJgCcz_px8Rj6WZpPZc5nyuRaSqF_Lf?usp=drive_link`
4. Run Backend: `python backend/app.py`
5. Launch Frontend: Open `frontend/index.html` in your browser.

Ethics & Compliance
This project is for defensive research purposes only. All data handling is compliant with **RA 10173 (Data Privacy Act of 2012)**.
